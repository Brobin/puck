# Puck Stats

A script to create a daily NHL team update for Reddit (in markdown). Primarily used to /r/wildhockey, but you can use it for whatever sub you want.

### Quickstart

 - Create a virtual environment
 - Install the requirements `pip install -r requirements`
 - Run the script with the argument `--team x` where `x` is the team id found in `puck/constants.py`
